# Task 1.3 — Backend Skeleton (Express + JWT)

**Phase:** 1 — MVP
**Status:** [x] Complete

## Objective
Build the Express application structure with all shared middleware, error handling, and routing scaffold.

## Steps

- [x] Scaffold `server/src/` directory structure:
  - `config/`, `middleware/`, `models/`, `controllers/`, `routes/`, `services/`, `utils/`, `constants/`, `templates/email/`
- [x] Install core dependencies:
  - express, helmet, cors, express-rate-limit, morgan, winston, joi, knex, pg, dotenv, bcrypt, jsonwebtoken, multer, file-type
- [x] Create `config/env.js` — Joi-validated environment variable loader (fail fast on missing required vars)
- [x] Create `config/db.js` — Knex instance with connection pool (`min: 2, max: 10`)
- [x] Create `middleware/errorHandler.js` — centralised error → `{ success: false, error: { code, message, details } }` JSON
- [x] Create `middleware/requestLogger.js` — Winston HTTP request logging (JSON format)
- [x] Create `middleware/rateLimiter.js` — per-route rate limit configs (auth: 10/15min, general: 100/min, export: 5/min)
- [x] Create `middleware/validate.js` — Joi schema validation factory (validates req.body/params/query)
- [x] Create `utils/asyncWrapper.js` — wraps async route handlers, forwards rejections to errorHandler
- [x] Create `utils/paginate.js` — standard pagination helper `{ data, pagination: { page, limit, total, totalPages } }`
- [x] Wire `app.js`: helmet, cors, json parser, rate limiter, request logger, routes, error handler
- [x] Create `server.js`: HTTP server with graceful shutdown on SIGTERM/SIGINT
- [x] Create `GET /api/health` endpoint returning `{ status, db, storage, timestamp }`
- [x] Add `routes/index.js` mounting all route modules under `/api/v1/`

## Acceptance Criteria
- Server starts and `/api/health` returns 200 with all services OK
- An intentionally thrown error returns the standard error JSON envelope
- Invalid JSON body returns 400 with a descriptive validation error

## Dependencies
- Task 1.1 (project structure), Task 1.2 (DB config)
